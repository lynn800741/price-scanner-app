LENSY 的萬物價格辨識實際上是如何運作的呢？
LENSY 的價格辨識使用革命性的 GPT-4o 人工智能技術。當您上傳物品圖片時，我們的 AI 會分析圖片中的所有細節，包括品牌、型號、材質、狀態等，並結合市場資料庫估算出合理的價格範圍。無論是商品、建築、自然景觀，甚至是太陽月亮，LENSY 都能給出創意而專業的估價。
LENSY 只接受當下拍的照片，還是可以從相簿中選取照片？
兩種方法都能在 LENSY 中使用。您可以即時拍攝想要辨識的物品，也可以從手機相簿中選取已存在的照片。在應用程式中，點擊中間的「相機」按鈕進入拍攝頁面，點擊白色圓圈即可拍照，或點擊左下方的圖片圖示從相簿選取照片。
我可以更改應用程式的語言嗎？
當然可以！LENSY 提供 16 種語言介面，包括繁體中文、簡體中文、英語、日語、韓語、泰語、越南語、西班牙語、法語、德語、義大利語、葡萄牙語、阿拉伯語、土耳其語、俄語和印尼語。更改語言的步驟：
  1. 開啟 LENSY 應用程式
  2. 點擊右下角「設定」圖示
  3. 在「支援」區塊中點擊「語言」
  4. 選擇您想使用的語言
如何查看我的會員資格是 LENSY Pro 會員或免費會員？
在 LENSY 應用程式的「設定」中，您可以查看會員狀態。請按照以下步驟操作：
  1. 開啟 LENSY 應用程式
  2. 點擊右下角「設定」圖示
  3. 在「會員服務」區塊查看「會員狀態」
  4. 您可以看到目前是「免費版」或「Pro 版」，以及剩餘掃描次數
如何在新的裝置中恢復我的 Pro 會員？
如果您已經創建 LENSY 帳戶並購買了 Pro 會員，只需在新裝置上使用相同帳戶登入即可。如果您是透過 Apple ID 或 Google 帳號訂閱，請確保：
  1. 使用相同的 Apple ID 或 Google 帳號登入
  2. 開啟 LENSY 應用程式
  3. 前往「設定」>「會員服務」
  4. 點擊「復原購買」
  5. 系統會自動恢復您的 Pro 會員資格
我需要註冊才能使用 LENSY 嗎？
是的，您需要註冊帳戶才能使用 LENSY 的完整功能。註冊帳戶的好處包括：
• 雲端同步您的掃描記錄
• 在不同裝置間無縫切換使用
• 確保資料安全，不會因更換手機而遺失
• 享受個人化的 AI 分析服務
• 獲得會員專屬優惠和功能更新

註冊非常簡單，只需使用電子郵件或透過 Apple/Google 帳號快速登入即可。
LENSY Pro 會員提供哪些福利？
LENSY Pro 會員解鎖完整的智能辨識功能，包括：
  1. 每月 1000 次無限制掃描分析
  2. AI 智能對話服務，深度了解物品資訊
  3. 批次照片辨識，一次處理多張圖片
  4. 匯出 Excel 報告功能
  5. 自動儲存掃描結果到相簿
  6. 去除所有廣告
  7. 優先客戶支援服務
  8. 未來新功能優先體驗
若物品辨識有誤，我該怎麼做？
由於 AI 辨識依賴圖片品質，請確保：
• 拍攝時光線充足
• 物品清晰對焦
• 包含完整的物品特徵

如果辨識結果不準確，您可以：
  1. 點擊結果頁面右上角的三點選單
  2. 選擇「更正物品識別」重新分析
  3. 或選擇「編輯名稱」手動修正
  4. 也可以透過「設定」>「聯絡我們」回報問題
我可以同時分析多張照片嗎？
可以！這是 LENSY Pro 會員的專屬功能。Pro 會員可以：
  1. 在相機頁面點擊左下角的圖片圖示
  2. 選擇多張照片（最多 10 張）
  3. 系統會依序分析每張照片
  4. 所有結果都會保存在歷史記錄中
如何更改價格顯示的貨幣？
LENSY 支援多種貨幣顯示，更改步驟：
  1. 開啟 LENSY 應用程式
  2. 點擊右下角「設定」圖示
  3. 在「支援」區塊中點擊「貨幣」
  4. 選擇您偏好的貨幣（如台幣、美元、人民幣等）
我的個人資料會有資安疑慮嗎？
您的隱私安全是我們的首要考量：
• 所有資料都經過加密處理
• 照片僅用於 AI 分析，不會儲存在伺服器
• 嚴格遵守 GDPR 和各國隱私保護法規
• 您可以隨時刪除本地儲存的所有資料
• 相機權限僅在您主動拍照時使用
如何關閉自動續訂或取消訂閱？
如果您想取消 LENSY Pro 訂閱，請按照以下步驟操作：

iOS 用戶：
  1. 開啟 iPhone 的「設定」
  2. 點擊您的 Apple ID
  3. 點擊「訂閱項目」
  4. 找到 LENSY 訂閱並點擊
  5. 選擇「取消訂閱」

Android 用戶：
  1. 開啟 Google Play 商店
  2. 點擊右上角的個人資料圖示
  3. 選擇「付款與訂閱」>「訂閱」
  4. 找到 LENSY 訂閱
  5. 選擇「取消訂閱」
如何刪除我的帳戶？
如果您想要刪除帳戶，請注意所有資料將永久刪除且無法恢復。刪除步驟：
  1. 開啟 LENSY 應用程式
  2. 前往「設定」>「帳號」
  3. 點擊「刪除帳戶」
  4. 確認刪除操作

注意：刪除前請確保已備份重要的掃描記錄。
如何聯絡客服支援？
如有任何問題或建議，歡迎透過以下方式聯絡我們：
  1. 應用程式內：「設定」>「聯絡我們」
  2. 電子郵件：lensyteam@gmail.com
  3. 回覆時間：48 小時內
